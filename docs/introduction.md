# Introduction

Yappa SDK is a modern, modular JavaScript/TypeScript library for real-time communication and event-driven applications. It provides robust connection management, event handling, and extensibility for building scalable client-side solutions.

- **Modular**: Use only what you need.
- **TypeScript-first**: Full type safety and autocompletion.
- **Production-ready**: Handles reconnection, heartbeats, and error management.
